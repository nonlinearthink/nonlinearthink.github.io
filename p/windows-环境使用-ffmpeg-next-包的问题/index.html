Windows 环境使用 ffmpeg-next 包的问题

Nov 03, 2024
阅读时长: 1 分钟

Issue

1

ERROR : failed to run custom build command for ffmpeg-sys-next v7.1.0

Solution

参考 Github Workflow，里面有 Windows 环境的具体编译脚本。

完整流程：

  1. 安装 LLVM 编译器

    使用 Visual Studio 安装 LLVM 编译器。

  2. 配置环境变量 LIBCLANG_PATH

    按照 Wiki 似乎可以直接加到 Path 变量里，但是我好像没有成功。

  3. 下载 FFmpeg

    https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-full-shared.7z

  4. 配置环境变量 FFMPEG_DIR

    填写你的 ffmpeg 解压后的路径就可以了

  5. 添加 path-to-ffmpeg\bin 到 Path 变量里

  6. 编译

    1

    cargo build
