Redux Persist 与 Provider

现在我们的 Redux 直接去用，还有一个问题，每次重新启动应用，会把数据重制，但是我们总有一些数据需要一直保存，比如代表登录信息的 token，这就是数据持久化。

Redux Persist 是一个社区的解决方案，需要配合 LocalStorage（react web）、AsyncStorage（react native） 之类的一起使用。

AsyncStorage 现在已经不在官方包里面了，已经交给社区去维护，所以需要 yarn add @react-native-community/async-storage。
